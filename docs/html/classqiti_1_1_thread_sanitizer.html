Detailed Description

Base class for ThreadSanitizer checks.

Provides a standardized API for generating different thread safety checks (e.g. running functions in a forked process with ThreadSanitizer enabled) and querying pass/fail status.

macOS-only: Factory to create a lock-order inversion detector.

When calling run(), tracks every mutex-acquire; if two locks are ever taken in inverted order on different threads, it flags failure.
