◆ getCallers()

std::vector< const FunctionData * > qiti::FunctionData::getCallers ( ) const
noexcept

Get all functions that have called this function.

Returns
A vector of pointers to FunctionData instances representing all functions that have called this function at least once during profiling. The vector may be empty if this function was only called from outside the profiled call stack.

Note: This only works reliably when you have called ScopedQitiTest::enableProfilingOnAllFunctions(true).
